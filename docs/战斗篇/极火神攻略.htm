极火神攻略

极火神，装等要求70，整体难度与极水神相当，途中会多次换T，需要Healer熟悉BOSS技能和自身走位，DPS需要知道在什么时候该做什么事，不要给Healer增加无谓的负担。开场玩家的初始位置在地图西面9点，BOSS在3点，请记好方向。或者事先放好区域标记，推荐将A放在1点右边，B放在2点，C放在4点。这样放的好处是，此后每次BOSS在施放完地狱之火炎后，都可以迅速找准位置进行躲避即将到来的光辉炎柱。总之，需要所有人(尤其是T和Healer)有足够的方向感，或者能够在关键时刻不跑错方向。初期开荒的话，有诗人在会比较好。

下面对BOSS技能先进行下说明：

烈焰焚烧：BOSS正面范围攻击，伤害1000左右，附带灼伤DEBUFF。

       ↑灼伤：根据DEBUFF积累程度，降低最大HP并增加其所受伤害。每增加1阶段，最大HP减少5%，所受伤害增加25%。需要在积累到2~3层时换T，换T方法如图。

火神爆裂：BOSS身边圆形范围，伤害较低，附带击退效果。

地火喷发：随机在玩家脚下生成裂痕，2秒后进行圆形范围攻击，伤害3000左右，一般瞄准随机2名DPS，每人3次，召唤火柱期间常瞄准ST。

地狱之火炎：BOSS瞬移到场地中间，根据场上存留的火柱数量，造成全图伤害。没有火柱时伤害为2000~3000，残留一根火柱约等于增加20000伤害，BOSS第4次使用地狱之火炎会导致强制性团灭。

灼热咆哮：轮流对两名Healer中的一人施放(如果期间没有死Healer的话)，初始伤害较低，附带灼伤DEBUFF。中了咆哮后，每隔5秒(15、10、5秒时)，会对该Healer身边生成圆形范围热风。

       ↑热风：处于热风中心的Healer并不会持续受到高伤害，但被卷入热风攻击范围的其他人会受到2000左右伤害，并附带击退效果。

光辉炎柱：BOSS在全图施放多个火圈，每个火圈伤害2000左右，均为圆形范围攻击。

↑※注意：BOSS只会在瞬移到场中心，放完地狱之火炎后，再使用光辉炎柱，届时全场将覆盖满火圈，只有两个对应的点不受覆盖。第一次为2点—8点，第二次为3点—9点，第三次为4点—10点，MT记得在地狱之火炎结束后提前将BOSS拉至对应的2、3、4点，而下一个即将被灼热咆哮瞄准的Healer则需要记得在地狱之火炎结束后提前移动至8、9、10点。

深红旋风：BOSS瞬移到场边分身成4体，逐个对前方进行大范围直线攻击，伤害3000左右。深红旋风只会在光辉炎柱结束后发动，届时除去中了灼热咆哮的Healer外，另外7人从MT一侧的第1只分身开始顺时针跑，被咆哮的Healer则需要从另外一侧的第1只分身消失处开始顺时针跑，躲不好的话硬吃一发也不至于被秒。

火狱之锁：BOSS召唤火柱后，会使用该技能对仇恨较低的T和随机一名DPS进行连线，被连线的两人会持续掉血，并根据距离降低两人伤害。连线期间DPS注意主动靠近连线的T，打其附近的火柱。

 

下面对BOSS的阶段变化和技能轴进行说明：(根据DPS不同，BOSS阶段变化的时间不同，所对应的技能也略有变化，但技能轴总体不变，意思也就是说这是个以技能轴为主的BOSS)

第一阶段：100%→80%HP

火神爆裂→烈焰焚烧→烈焰焚烧→灼热咆哮→地火喷发→火神爆裂→烈焰焚烧x3→灼热咆哮

MT开怪后转身，让BOSS背对队友。开始后不久，第一次灼热咆哮就会丢给Healer中的一人，中了咆哮后主动跑向12点位置靠边站着就行，可以事先做好20秒的计时宏(其实15秒就够，最后5秒结束不会有热风了)，待咆哮DEBUFF结束后回到BOSS身后，下一次就该轮到另一个Healer跑往12点位置等咆哮结束了。其余人躲BOSS技能时不要往12点附近跑，不然吃到热风伤害会很疼，无谓增加Healer负担。此后战斗的重点都是以Healer轮流躲好咆哮为基础，团灭的大多数原因就是这个。两个T可以互相FOCUS对方，关注对方身上的灼伤DEBUFF，一般3层必须换T了，越早换T，Healer压力越小。不过由于BOSS的正面范围攻击夹角比较大，在换T时容易打到其它人，所以，如果ST从BOSS左侧挑衅切入的话，MT最好右侧离开(记得看到ST挑衅技能效果，自己就收手关攻击吧)。而Healer也要时刻关注MT的更换。

 

第二阶段：(第1次召唤火柱→第1次地狱之火炎，(HP 80%→79%，为什么HP变化这么少？因为你们都去打柱子了= =)

第1次召唤火柱(4根)→灼热咆哮→地火喷发→烈焰焚烧→火神爆裂→烈焰焚烧x2→灼热咆哮→烈焰焚烧→地火喷发→第1次地狱之火炎

第1次只有4根火柱，很好清理，12点位置的火柱交给队里远程清理，或者清理时让被咆哮的Healer暂时往南移动下，让开12点位置。本阶段没什么难的，主要是末尾的第1次地狱之火炎结束后，没有被前一次咆哮所瞄准的Healer要提前跑到2点对面的8点位置，躲避在第1次光辉炎柱后即将到来的这次咆哮，否则，马上全场会覆盖火炎炎柱的攻击范围，要被咆哮瞄准的Healer就来不及跑到对面8点位置躲咆哮了，要死自己死掉，要么导致队友卷入热风中。

 

第三阶段：第1次地狱之火炎结束→第2次召唤火柱，(HP 79%→50%)

烈焰焚烧→第1次光辉炎柱→灼热咆哮→烈焰焚烧x2→第1次深红旋风→灼热咆哮→地火喷发x3→烈焰焚烧x3→灼热咆哮→地火喷发x3→烈焰焚烧x2

当看到BOSS分身成4体后，中了咆哮的Healer身上的效果应该还未消失，需要在另外一边半场躲避(如果你没有延迟的话其实挺好躲的)。其余7人集中在MT一侧一起顺时针靠墙跑即可。躲完后MT继续回到3点位置等BOSS出现。

 

第四阶段：第2次召唤火柱→第2次地狱之火炎，(HP 50%→49%)

第2次召唤火柱(8根)→烈焰焚烧→灼热咆哮→烈焰焚烧→地火喷发→烈焰焚烧→烈焰咆哮→烈焰焚烧x2→火神爆裂→烈焰咆哮→烈焰焚烧→第2次地狱之火炎

第2次8跟火柱对全队来说DPS应该也不成问题，推荐DPS从4点位置的柱子开始，贴着外围火圈顺时针清火柱，清到12附近时，被咆哮的Healer要主动往南移动，腾出12点位置让DPS清火柱。被火狱之锁连线的两人注意主动靠近打同一根柱子，如果期间被连线的ST需要去换跟MT交换BOSS，被连线的DPS注意跟着ST过去，然后打BOSS脚下那根柱子。末尾的第2次地狱之火炎结束后，下一次要被咆哮瞄准的Healer记得提前跑到9点位置。

 

第五阶段：第2次地狱之火炎结束→第3次召唤火柱，(HP 49%→30%)

烈焰焚烧→第2次光辉炎柱→灼热咆哮→烈焰焚烧x2→第2次深红旋风→灼热咆哮→地火喷发x3→烈焰焚烧x3→灼热咆哮→地火喷发x3→烈焰焚烧x2

技能轴跟之前基本一样，只需要注意第2次光辉炎柱的躲避位置是3点和9点。下一次要被咆哮瞄准的Healer在第2次地狱之火炎结束后，提前跑到9点位置，其余人集合在3点位置顺时针跑。

 

第六阶段：第3次召唤火柱→第3次地狱之火炎，(HP 30%→29%)

第3次召唤火柱(13根)→烈焰焚烧→灼热咆哮→烈焰焚烧x2→地火喷发→火神爆裂→烈焰焚烧→灼热咆哮→烈焰焚烧x3→地火喷发→灼热咆哮→烈焰焚烧x3→地火喷发→灼热咆哮→烈焰焚烧x3→火神爆裂→灼热咆哮→烈焰焚烧→地火喷发→烈焰焚烧x2→灼热咆哮→第3次地狱之火炎

第3次召唤的火柱数量比较多，不过从召唤火柱到第3次地狱之火炎的时间间隔也比较长，中间穿插的技能比较多。如果有黑魔或者召唤的话最好丢下极限技，实在没有让近战DPS丢个单体的弥补下DPS也比憋着不放强。期间被咆哮的Healer注意给DPS让位置清理火柱，被连线的两人记得靠近，来不及换T就开无敌减伤吧，只要撑过第3次地狱之火炎后，队里没有大量减员，通关基本不成问题了。

 

第七阶段：第3次地狱之火炎结束后→第4次地狱之火炎引发强制性团灭，(HP 29%→0%)

烈焰焚烧→第3次光辉炎柱→灼热咆哮→烈焰焚烧x2→第3次深红旋风→灼热咆哮→地火喷发x3→烈焰焚烧x3→第4次光辉炎柱→灼热咆哮→烈焰焚烧x2→第4次深红旋风→灼热咆哮→地火喷发x3→第4次地狱之火炎(强制性团灭)

注意第3次光辉炎柱的躲避位置是4点和10点，下一次要被咆哮瞄准的Healer需要在第3次地狱之火炎结束后，提前跑到10点位置，其余人集合在4点位置顺时针跑。DPS充足的话，一般第3次深红旋风结束后不久就可完成击杀。

 

总结：

1.       团灭主要原因之一是Healer不知道自己被咆哮后该怎么应对，以及没掌握好自己被咆哮的顺序，导致队友吃到了热风，造成减员；

2.       团灭主要原因之二是Healer没注意看好换T后的MT的HP；

3.       团灭主要原因之三是不知道那个覆盖全图的光辉炎柱的躲避位置和位置的变化顺序；

4.       团灭主要原因之四是清理火柱时在12点位置躲避咆哮的Healer没有往南移动，没有给DPS腾出清理火柱的位置，或者在走位时将队友卷入了热风，导致减员；

5.       团灭主要原因之五是两名T之间换手BOSS没交接好。
