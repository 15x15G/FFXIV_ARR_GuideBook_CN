50副本纷争要地 布雷福洛克斯野营地

装等要求55，2.2添加的新副本，将会是大部分玩家之后刷的最多的一个副本，是一个刷神话较理想的副本，刷神话一般只打前两个BOSS。出产一些70级的装备，以及一个非常可(zhi)爱(qian)的迷你宠物，小奥猴。装等刚达标的4人也不是太难，只要注意MT把怪拉到不容易引到其它怪的地方击杀，DPS和Healer不要比MT跑的太靠前，保持跟MT打同一个目标就好。

PS：仿佛已经看到了国服里会出现一些为了刷神话而去自动排本进副本，打过俩BOSS后就马上退队的玩家身影了……

一开始途中打怪时注意躲避落下的炸弹，圆形范围伤害。另外注意解除麻痹状态。

BOSS战1：

注意躲避BOSS的炸弹的攻击范围，刷了小怪DPS尽快解决就好，DPS很高的话也可以MT拉着小怪仇恨集火直接打BOSS。

BOSS战2：

会时不时对正面进行范围攻击，DPS别站BOSS脸上。

会定期刷小怪，DPS注意优先清理，不要考虑小怪仇恨，最好能不要将小怪透支的炸弹诱导至队友所站的那块蓝池上。

后半段会施放多个蓝色圆形区域，站在蓝池里受到小怪投掷的炸弹攻击会诱发爆炸，受到更高伤害，站在两个蓝池的重叠区域诱爆伤害也会翻倍。

BOSS偶尔会无视仇恨的随机对其他玩家进行攻击。

BOSS快死前会施放全屏攻击，站在蓝池上会吃到大量伤害，站在两块蓝池重叠的区域除了满血的MT外基本必死。

最终BOSS战：

BOSS每减少一定HP，就开始绕圈，并同时刷两个小怪，赶快清理小怪，因为BOSS绕圈停下来后便会开始往场中心扔出大量炸弹，根据玩家攻击技能的施加方向，炸弹会被弹向另一边，全员集中在场地中心将中心的炸弹都打到场地外围即可。

此后将会重复上述模式数次，待BOSS只剩下最后一点HP时，会往场地中心丢一颗超大的炸弹，如果有极限技的话，可以让DPS放极限技在爆炸前打掉BOSS。如果没有极技或DPS不足，需要先打掉炸弹，再输出BOSS。
