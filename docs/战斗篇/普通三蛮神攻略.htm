普通三蛮神攻略(火神、岩神、风神)

一、         普通火神 C 伊芙利特

玩家们在主线任务到20级时将会第一次与蛮神遭遇，不过第一次遇到的蛮神均为普通难度，除了技能种类少、伤害低外，副本也限制为4人队伍。普通火神伊芙利特的所有读条技能均可以被物理系的眩晕类技能击晕打断，同时可听到FinePlay的效果音。

Tank开拉BOSS后，迅速转身，自己背靠3点火墙，使BOSS背对其他队友，因为BOSS会频繁使用正面大角度的喷火攻击。战斗可尽量多打断BOSS的读条技能，来增加极限技能槽的积蓄速度，而未被打断的技能则要躲避好，连续吃两三发地火喷发或光辉炎柱也是很容易死的。

当BOSS的HP下降到50%左右时，将会召唤一根炎柱，全员(包括T和Healer)都要优先对炎柱进行攻击，否则一定时间内炎柱没有被破坏掉的话，BOSS会施放秒杀全屏玩家的团灭技能。玩家们手里那些CD较长的高伤害技能可以留到炎柱出现后再用。

 

二、         普通岩神 C 泰坦

在主线任务进行到34级时，将第一次遭遇泰坦，普通泰坦依然为4人副本，技能种类少，伤害低、模式简单。

Tank开拉BOSS后，依然迅速转身，自己背靠悬崖边缘，使BOSS背对其他队友。战斗期间BOSS会经常回身蓄力进行带有蓝色技能范围提示的直线攻击，高伤害并伴有强击退效果，距离悬崖边较近时则会被击退落入悬崖，玩家要注意躲避。

当BOSS的HP下降到一定量时，会先后两次跳至高空砸落下来，造成全屏伤害的同时，会震落掉红色技能提示区域的地板，站在红色技能提示区域中的玩家将会落入悬崖，并处于无法复活状态，所以玩家一定要注意BOSS跳起时不能站在红色区域里。

同时，战斗期间BOSS会使用花岗岩牢狱禁闭一名玩家，其他队友要优先帮队友破坏掉石笼，打石笼的同时要注意躲避来自BOSS的范围攻击。

当BOSS最后一次跳起落下后，会露出新目标-泰坦的核心，所有玩家必须优先攻击核心，否则一定时间过后，泰坦将会施放大地之怒，在核心未破坏掉的状态施放大地之怒，将会导致全屏秒杀。推荐DPS职业把极限技丢给泰坦的核心。

 

三、         普通风神 - 迦楼罗(介于很多同学第一次接触，而且该BOSS又比之前的难一些，特别写详细点，各职业玩家取自己所需攻略点即可)

玩家在主线任务进行到44级时，将第一次遭遇风神，普通风神虽然为4人副本，但相比普通火神和岩神来说，难度一下子提升了不少，但总体难度依然是偏简单的。

优先推进主线任务进度的玩家，也许在44级的时候就可以匹配风神讨伐战了，考虑到游戏初期开荒的难度，以及45级对角色能力和装备的提升，更推荐45级完成了职业任务后再来匹配。如果4人都是44级的纯开荒队的话，团灭3,、4次恐怕都是很正常的。

首先对战斗场景做下说明：

玩家一开始出生在6点位置，BOSS在北面12点，BOSS背后有一个背景石堆，不会被破坏，可作为12点的方向标供玩家参考，同样6点玩家出生点背后，也有一面旗帜可作为方向标参考。如下图。

  

如果玩家害怕在战斗中失去方向感，也可以在战斗前提前放下ABC的区域标记，战斗中跑错方向，往往会使自己或团队陷入团灭节奏。

场地中央有4个石柱，是确保玩家生存、用来降低BOSS全屏攻击技能伤害用的，但耐久度相对于真、极风神的石柱低很多，也就是说普通的石柱很脆弱，这点需要Tank注意。

下面对迦楼罗施放的主要技能做下说明：

下行突风：BOSS正面近距离范围攻击，瞬发，打到非T职业玩家会比较痛，场地中的石柱也很容易因T走位不慎，而被此技能打到破坏掉。T记得开怪后立刻将BOSS拉到12点石堆，使BOSS面向12点石堆。

螺旋气流：BOSS正面直线范围攻击，伤害较高并附带眩晕效果，有较长读条时间，会瞄准仇恨最高者施放，T需要在放出技能前，穿过BOSS走位到其身后，待技能放完后再回到原位。

邪轮旋风：以BOSS为中心，对周围进行圆形范围攻击，瞬发，伤害较高。所以推荐其他玩家站位不要太靠近T。

烈风刃：BOSS远距离范围攻击，瞄准随机T以外的目标。推荐其他玩家彼此之间，拉开一定距离，分散站位，而且不要站在BOSS和另一名玩家之间的直线路径上。

寒风之歌：BOSS消失后，会瞬移到12点施放此技能(下次会瞬移到6点位置施放)，直线范围攻击，有较长读条时间，伤害2000HP左右，初期躲在石柱背后可躲掉伤害。

↑在战斗的最后阶段，BOSS会在台风眼中心瞄准随机玩家施放寒风之歌，此时场地上已没有石柱可供躲避，但BOSS读条时不会改变施放技能的方向，被瞄准和可能被打到的玩家看到BOSS读条时要及时绕到BOSS背后来躲避最后一次寒风之歌。

召唤刚羽：BOSS施放寒风之歌后，会召唤一批刚羽，刚羽从场地周围出现，不会攻击玩家，但会自动向石柱聚拢，一定时间后施放羽枪自爆，对周围小范围造成200左右的伤害，放任不管的话自爆会破坏掉石柱，导致后面全屏技能时无法减伤而团灭。黑魔的范围攻击和白魔45的神圣可轻松解决。

大气爆发：DPS正常的话，一般在BOSS的HP到达50%左右时，会施放该技能，全屏高伤害攻击，根据场上石柱的存留根数和耐久，降低此时玩家所受伤害。如果石柱存留情况比较悲剧，也可以开启T的极限技撑过去。

台风眼：BOSS施放大气爆发后，会在场地上召唤一个台风眼。台风眼会一直存在到战斗结束，玩家的可活动范围也会被限制在台风眼中心的较小圆形区域，踏入守卫的风暴中会受到伤害和强击退效果，将玩家弹回场地中间。台风眼的出现标志着战斗进入了后半段。

寒风之啸：在战斗的最后阶段(BOSS的HP约10%HP左右时)，BOSS会瞬移到12点位置，读条施放一次寒风之啸，伤害非常高，很容易导致团灭。技能范围及躲避方法图示可参考下图。

技能介绍完后，最后来说下主要技能的施放顺序：(如果团队的DPS一直存活，且输出正常的话)

第1次瞬移到12点施放寒风之歌 → 第1次寒风之歌后会立刻召唤第1批刚羽 → 第2次瞬移到6点施放寒风之歌 → 第2次寒风之歌后会立刻召唤第2批刚羽 → (下接)

→ 第3次瞬移到场地中间，同时召唤第3批刚羽，但并不会进行什么特别的攻击技能 → 第3批刚羽清理完不久(BOSS的HP50%左右时)会进行大气爆发 → 大气爆发后场上石柱消失，同时生成台风眼限制玩家活动区域 → (下接)

→ BOSS的HP10%左右时，将瞬移到12点施放寒风之啸 → 寒风之啸结束后不久BOSS会瞬移到场地中间，瞄准随机玩家施放寒风之歌 → 如果DPS充足的话此后不久便能击杀掉BOSS，否则将按以下顺序循环施放技能 → (下接)

→ 召唤刚羽 → 瞬移到12点施放寒风之啸 → 召唤刚羽 → 瞬移到场地中间施放寒风之歌 →…………

最后总结下普通风神容易引发团灭的因素：

1.       T没拉好BOSS，或拉BOSS的位置不对，或有人OT，导致BOSS的攻击误伤到石柱，从而引发后面扛不住大气爆发，陷入团灭；

2.       刚羽没及时清理掉，导致石柱被破坏，使得后面扛不住大气爆发的伤害，从而陷入团灭；

3.       T不会卡视角躲BOSS的螺旋气流，导致自己在眩晕的时候被OT，或者由于吃到螺旋气流的伤害，导致Healer没来得及加上，而倒T；

4.       战斗最后阶段的寒风之啸和此后场地中间的寒风之歌，没有顺利躲掉，导致减员，陷入团灭。
